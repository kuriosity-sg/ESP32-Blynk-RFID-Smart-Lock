# ESP32-RFID-Smart-Lock
Using an ESP32 module, an RFID reader and a Solenoid lock, the ESP32 App lock projectÂ allows you to lock and unlock a Smart Lock with an RFID smart card or via an app on the cloud.

## Hardware Required
- WIFI Bluetooth Module ESP32-WROOM-32E
- Relay Module 1 Channel 5V
- RFID / NFC Module PN532 13.56MHz
- Power Adapter 12V 2.5A Safety Mark
- Jumper Wire Male-Male 40x
- Solenoid Door Lock
